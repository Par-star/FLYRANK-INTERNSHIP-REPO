# 🚀 FlyRank ML Internship – Search Ranking & Content Refresh Prediction

This repository contains my work completed as part of the **FlyRank Machine Learning Internship**.

The internship focuses on applying practical Machine Learning techniques to solve real-world search ranking and content optimization problems using structured datasets and interpretable models.

---

## 📚 Repository Overview

The repository includes weekly notebooks that progressively build an ML workflow:

- **Week 1:** Data exploration and discovery
- **Week 2:** Building interpretable machine learning models
- Model evaluation
- Ranking pages for content refresh
- Feature engineering
- Performance comparison against rule-based baselines

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/<your-username>/<repo-name>.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Launch Jupyter Notebook

```bash
jupyter notebook
```

Run notebooks in sequence:

1. `01_first_look_and_discovery.ipynb`
2. `02_your_first_readable_model.ipynb`
